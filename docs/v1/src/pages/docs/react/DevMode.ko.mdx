## `<DevMode/>`

<Callout type="warning">

DevMode is experimental feature, this interfaces could be changed

</Callout>

### Setup

After setup DevMode, We can use devMode prop in each component

```jsx /DevMode/
import { Suspensive, SuspensiveProvider, DevMode } from '@suspensive/react'

const App = () => {
  const [suspensive] = useState(() => new Suspensive())

  return (
    <SuspensiveProvider value={suspensive}>
      {/** children can use devMode now */}
      <DevMode />
    </SuspensiveProvider>
  )
}
```

#### `<Suspense/>` devMode prop

```jsx /devMode/
import { Suspense } from '@suspensive/react'

const Example = () => (
  <Suspense
    fallback={<>loading...</>}
    devMode={{
      // devMode prop will work only in development mode
      showFallback: true,
    }}
  >
    children
  </Suspense>
)
```

#### `<ErrorBoundary/>` devMode prop

```jsx /devMode/
import { ErrorBoundary } from '@suspensive/react'

const Example = () => (
  <ErrorBoundary
    fallback={({ error }) => <>{error.message}</>}
    devMode={{
      // devMode prop will work only in development mode
      showFallback: true,
    }}
  >
    children
  </ErrorBoundary>
)
```

```jsx /devMode/
import { ErrorBoundary } from '@suspensive/react'

const Example = () => (
  <ErrorBoundary
    fallback={({ error }) => <>{error.message}</>}
    devMode={{
      // devMode prop will work only in development mode
      showFallback: { after: 2000 },
    }}
  >
    children
  </ErrorBoundary>
)
```
