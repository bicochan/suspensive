# SuspensiveProvider

이 컴포넌트는 Suspensive가 제공하는 컴포넌트들(Suspense, Delay 등)의 설정을 한 번에 제어합니다.

```tsx /SuspensiveProvider/ /Suspensive/
import { Suspensive, SuspensiveProvider } from '@suspensive/react'

const suspensive = new Suspensive({
  defaultOptions: {
    delay: {
      ms: 1200,
    },
    suspense: {
      fallback: <Spinner />,
    },
  },
})

function App() {
  return <SuspensiveProvider value={suspensive}>...</SuspensiveProvider>
}
```

## SuspensiveDevMode

<Callout type="warning">

devMode is experimental feature, this interfaces could be changed

</Callout>

### Setup

```jsx
import { Suspensive, SuspensiveProvider, SuspensiveDevMode } from '@suspensive/react'

const App = () => {
  const [suspensive] = useState(() => new Suspensive())

  return (
    // devMode work only with `<SuspensiveProvider/>`
    <SuspensiveProvider value={suspensive}>
      {/** children can use devMode now */}
      <SuspensiveDevMode />
    </SuspensiveProvider>
  )
}
```

#### `<Suspense/>` devMode

```jsx /devMode/
import { Suspense } from '@suspensive/react'

const Example = () => (
  <Suspense
    fallback={<>loading...</>}
    devMode={{
      // devMode prop will work only in development mode
      showFallback: true,
    }}
  >
    children
  </Suspense>
)
```

#### `<ErrorBoundary/>` devMode

```jsx /devMode/
import { ErrorBoundary } from '@suspensive/react'

const Example = () => (
  <ErrorBoundary
    fallback={({ error }) => <>{error.message}</>}
    devMode={{
      // devMode prop will work only in development mode
      showFallback: true,
    }}
  >
    children
  </ErrorBoundary>
)
```

```jsx /devMode/
import { ErrorBoundary } from '@suspensive/react'

const Example = () => (
  <ErrorBoundary
    fallback={({ error }) => <>{error.message}</>}
    devMode={{
      // devMode prop will work only in development mode
      showFallback: { after: 2000 },
    }}
  >
    children
  </ErrorBoundary>
)
```
